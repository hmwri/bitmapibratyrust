# bitmapibratyrust

## 日本語

Rust で BMP/ビットマップデータを扱うための実験リポジトリです。`Cargo.toml` 上の crate 名は `bmp` で、現状は外部依存なしの小さなプロトタイプです。

### 構成

- `Cargo.toml`: crate 設定
- `src/`: Rust source

### ビルド

```bash
cargo build
```

### 実行

```bash
cargo run
```

### 注意

公開されている tree だけでは CLI 仕様が明確ではないため、README では BMP 処理実験として記述しています。具体的な入出力仕様を決めたら追記してください。

## English

A Rust experiment for working with bitmap/BMP data. The crate is named `bmp` and currently has no external dependencies.

### Build

```bash
cargo build
```

### Run

```bash
cargo run
```
